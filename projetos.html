Desenvolvimento de sistemas de controle automatizado

Sistemas simples de controle

A característica principal desses sistemas é a presença de rele programável. As principais vantagens do rele programável em comparação com o rele de contato é a robustez, dimensões reduzidas e custo. Além disso o próprio programa do rele foi criada em linguagem logica o que permite começar trabalhar quase que imediatamente sem necessidade de ter a experiência de programação dos controladores lógicos. Geralmente os sistemas descritos se baseiam em relê EATON Easy.

Sistemas complexos de controle

Esses sistemas de controle são destinados as empresas com alto nível de conhecimento dos sistemas de controle automatizado de produção. Esses sistemas são baseados em controladores lógicos programados.

Os sistemas de controle podem ser projetados para níveis 0-2 de automatização. Principais características:

  • Programação em linguagens IEC61131-3 (LAD/LD, FBD, STL/IL, SCL/ST);
  • Aplicação da periferia distribuída e decentralizada;
  • Aplicação de Profibus-DP para os sistemas industriais simples;
  • Industrial Ethernet/Wi-Fi no nível físico do modelo OSI para as redes industriais mais complexas. Os outros protocolos de modelos físicos aplicados (por exemplo, Profinet) podem ser aplicados com os protocolos físicos mencionados;
  • Capacidade de interagir com o nível 3 de automatização (MES);
  • O sistema de emergência corresponde ao necessário SIL/SIL CL, PL;
  • Como modulo para esses sistemas usa-se o Siemens SIMATIC PLC S7-300/S7-1500;

O motor elétrico moderno

Para o controle efetivo e o gerenciamento do processo com os sistemas de controle mencionados são usados os conversores de frequência (CF). A interação pode ser realizada tanto por meio de entradas e saídas discretas, como, por exemplo, para um sistema de controle simples ou por meio de uma rede industrial para sistemas avançados. A complexidade do circuito CF pode variar de simples (usando um modo de controle escalar para um par de motor para conversor) para complexo (usando um acoplamento mestre-escravo para vários transdutores no modo de controle vetorial) dependendo da tarefa a ser realizada.

A solução padrão para essa unidade elétrica é o uso de CF ABB, Schneider, Danfoss.

Interface Homem-Maquina (IHM)

Pode-se desenvolver a IHM para os sistemas de controle acima descritos. As IHMs podem ser divididas em seguintes grupos:

  • IHM primitivo que é composto apenas por sistemas de sinalização e estrutura para introduzir comandos;
  • IHM primitivo com uma tela monocromática de baixa resolução;
  • IHM com painel do operador (Siemens SIMATIC HMI Comfort TP). Pode ser usado para comandar os sistemas de controle mais elaborados;
  • IHM baseada em sistemas SCADA. Geralmente necessita de um computador e SIMATIC WinCC RT v.1x. Pode ser usado para controle de sistemas de alta complexidade;
