Design of control systems

Simple control systems

The main characteristic of these systems is the presence of programmable relay. The main advantages of programmable relay compared to contact relay are robustness, reduced dimensions and low cost. In addition, the relay program itself was created in a logical language, which allows starting working almost immediately without any previous experience in the programming of the logic controllers. Generally, the systems described are based on EATON Easy relay.

Complex control systems

These control systems are designed for companies with a stuff that has a high level of knowledge of automated control systems. These systems are based on programmable logic controllers.

Control systems can be designed for automation levels 0-2. Main features are:

  • Programming in languages ​​IEC61131-3 (LAD/LD, FBD, STL/IL, SCL/ST);
  • Application of distributed and decentralized periphery;
  • Application of Profibus-DP for simple industrial systems;
  • Industrial Ethernet / Wi-Fi at the physical level of the OSI model for the more complex industrial networks. The other protocols of applied physical models (for example, Profinet) can be used with the physical protocols mentioned above;
  • Ability to interact with level 3 automation (MES);
  • As a module for these systems the Siemens SIMATIC PLC S7-300 / S7-1500 is used;

Modern electric motor

Frequency converters (CF) can be used in order to provide an effective control and management of the process using control systems mentioned above. The interaction can be performed either by means of discrete inputs and outputs, for example for a simple control system or through an industrial network for advanced systems. The complexity of the CF circuit can range from simple (using a scalar control mode to a converter motor pair) to complex (using a master-slave coupling to multiple transducers in vector control mode) depending on the task to be performed.

The standard solution for this electrical unit is the use of CF ABB, Schneider, Danfoss.

Human Machine Interface (HMI)

The HMI can be developed for the control systems described above. HMIs that can be designed may be divided into the following groups:

  • Primitive HMI which consists only of signaling systems and features for commands entering;
  • Primitive HMI with a low resolution monochrome display;
  • HMI with operator panel (Siemens SIMATIC HMI Comfort TP). Can be used to control more complex control systems;
  • HMI based on SCADA systems. You usually need a computer and SIMATIC WinCC RT v.1x in order to provide its functionality. Can be used to control high complexity systems;
